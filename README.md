# Проект "Поиск вакансий"

## Описание:
Данный проект - это программа, которая будет получать информацию о вакансиях с платформы hh.ru в России, сохранять ее в файл и позволять удобно работать с ней: добавлять, фильтровать, удалять.

## Установка:
### 1. Клонируйте репозиторий: 

        git clone https://github.com/basinanatali-star/project_folder
### 2. Установите зависимости:

        pip install -r requirements.txt

## Использование:
Это учебный проект, поэтому можно использовать его как образец.  

## Тестирование:
Тестирование кода играет важную роль в разработке программного обеспечения. Оно не просто помогает обнаруживать ошибки и проблемы в коде до того, как продукт будет выпущен, но также способствует высокому качеству и надежности программных решений.
    
### Причины важности тестирования:
* Обеспечение качества и надежности.
* Повышение уверенности при внесении изменений.
* Ускорение процесса разработки.
* Обеспечение совместимости и соответствия.

### Модули проекта:
* Модуль, содержащий абстрактный класс для работы с API сервиса с вакансиями.
* Модуль, содержащий класс для работы с вакансиями, поддерживающий методы сраввнения вакансий по зарплате.
* Модуль, содержащий абстрактный класс, который реализовыввает методы для добавления вакансий в файл, получения данных из файла по указанным критериям и удаления информации о вакансиях.
* Модуль, содержащий функцию для взаимодействия пользователя и объединяющий все классы и функции в единую программу, доступ к которой можно получить из основного файла main.py.
* Модули, содержащие тесты ко всем функциям.
